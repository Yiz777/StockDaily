## 🔭 机会雷达规则（v3 — 持续追踪模式）

### 核心理念
**好的机会不会每天出现。专业分析师维持一个"观察清单"持续追踪，而不是每天发明新机会。**

### 运行机制

生成日报前，读取 `0. Newsletter\watchlist.md`（观察清单）。

如果 watchlist.md 存在：
1. 更新已有条目的最新状态（今天的新闻/数据有没有影响这个方向）
2. 只在发现真正新的、值得追踪的方向时才添加新条目
3. 已兑现或已失效的条目标记状态变更

如果 watchlist.md 不存在或为空：
1. 尝试发现1个值得长期追踪的新方向
2. 创建 watchlist.md

### watchlist.md 格式

```markdown
# 🔭 观察清单

## 🔄 半导体周期（非AI芯片） — 追踪中
- **类型**：🔄 周期定位
- **开始追踪日期**：2026-06-27
- **状态**：🟡 追踪中
- **逻辑**：德州仪器Q1营收+19%，库存周期见底信号
- **催化剂**：Q2财报季（7月中旬）
- **更新**：2026-07-02 三星暂停接单，产能紧缺佐证需求
- **怎么参与**：SOXX ETF / TXN个股（需完整财务数据）

## 📊 小盘股（Russell 2000） — 追踪中
- **类型**：🔄 周期定位
- **开始追踪日期**：2026-07-02
- **状态**：🟡 追踪中
- **逻辑**：Q2涨21%，资金从大科技轮动到小盘
- **催化剂**：非农确认加息降低（已兑现✅）
- **更新**：2026-07-02 非农+5.7万，加息预期大降，小盘股受益
- **怎么参与**：IWM ETF
```

### 状态标签

| 状态 | 含义 |
|---|---|
| 🟢 新发现 | 刚加入清单，需要更多数据验证 |
| 🟡 追踪中 | 逻辑成立，等催化剂兑现 |
| 🔵 催化剂兑现 | 等待的信号已出现，升级为"值得行动" |
| ✅ 已兑现 | 机会已经反映在价格中，可以移除 |
| ❌ 逻辑失效 | 基本面恶化或判断错误，标记原因后移除 |

### 每日日报里机会雷达板块的写法

**情况1：有正在追踪的条目（最常见）**
```
🔭 机会雷达

🔄 半导体周期（非AI芯片）
状态：🟡 追踪中（第X天）
今日更新：[今天的新闻/数据对这个方向有什么影响？]
[如果今天没有新信息影响它，写"今日无新变化，继续追踪"]

📊 小盘股
状态：🟡 追踪中（第X天）
今日更新：[同上]
```

**情况2：今天发现了新机会**
```
🔭 机会雷达

🟢 新方向：[新方向名]
[完整分析：为什么值得关注/数据信号/怎么参与]
→ 已加入观察清单

🔄 [已有条目1]
状态：🟡 追踪中
今日更新：...
```

**情况3：今天没有新发现，追踪的条目也无变化**
```
🔭 机会雷达

今日无新方向。正在追踪：
🔄 半导体周期 — 🟡 追踪中（第6天）
📊 小盘股 — 🟡 追踪中（第3天）

耐心等待催化剂。不是每天都需要新机会。
```

### 引入新方向的门槛（严格）

每周最多引入1-2个新方向。新方向必须满足以下全部条件：

**必须回答三个问题：**
1. **为什么是现在？**（不是上周，不是下个月，为什么今天加入观察清单）
2. **数据信号是什么？**（具体哪个数字变了，不是"感觉值得关注"）
3. **和读者已有的持仓有什么不同？**（如果VOO/QQQ里已经有了，不配进清单）

**必须满足至少一个引入依据：**

#### 依据A：数据拐点
某个具体指标发生了方向性变化。
- ✅ "德州仪器Q1营收+19%，连续2季度加速，库存天数从80天降到55天"
- ❌ "半导体行业可能复苏"

#### 依据B：重大事件创造新机会
某个事件改变了某个行业/资产的基本面逻辑。
- ✅ "特朗普政府要求限制GPT-5.6发布→AI监管可能利好隐私计算赛道"
- ❌ "AI概念很热"

#### 依据C：估值错配
好东西被低估了——价格跌了但基本面没坏。
- ✅ "迪士尼从高点跌30%，但Disney+用户增速回到+8%，P/E 15x低于5年均值22x"
- ❌ "迪士尼跌了不少，可能值得买"

**必须排除的假信号：**
- ❌ 今天暴涨的股票（市场已经知道了，你进场是接盘）
- ❌ 某分析师说"强烈推荐"（没有自己的数据判断）
- ❌ "长期来看XX一定涨"（没有时间窗口，等于什么都没说）
- ❌ 已经在VOO/QQQ里占大权重的方向（你已经有了）
- ❌ 上周刚推过然后移除的（死了的方向不要复活，除非有新数据）

### 什么时候移除条目

| 情况 | 操作 |
|---|---|
| 催化剂已兑现（如财报确认周期反转） | 标记🔵→✅，保留1周后移除 |
| 逻辑失效（如基本面恶化） | 标记❌，写明原因，保留1周后移除 |
| 追踪超过3个月仍无催化剂兑现 | 评估是否值得继续追踪，可能需要移除 |
