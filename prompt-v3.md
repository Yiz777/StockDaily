## Step 1.6：⚠️ 盘后异动检测（绝不跳过）

**这一步是 v3 新增。跳过 = 严重失职。历史上出现过"收盘涨2%但盘后暴跌8%，日报却写'表现不错'"的事故。**

### 操作步骤：
1. 对比每个标的的「收盘价」和搜索到的「盘后/期货价格」
2. 计算盘后变动幅度
3. 触发阈值：

| 标的类型 | 盘后异动阈值 | 触发后行动 |
|---|---|---|
| 指数（标普/纳指/道指） | 盘后波动 > ±1% | 写入"今日一句话" + 温度计卡片加盘后行 + 调整"今日建议" |
| 持仓个股 | 盘后涨跌 > ±3% | 写入"持仓预警"板块，标红，标注盘后变动幅度和原因 |

### 盘后异动在日报中的显示方式：

**1. Header 的 badge**：加红色badge `🚨 盘后异动：纳指期货-X%！`

**2. 温度计卡片**：在收盘价下方加一行小字：
```
📊 标普500    7,483  -0.22%
  └ 盘后期货 -2.3% ⚠️
```

**3. 今日一句话**：必须综合收盘价和盘后走势。如果盘后方向与收盘相反，必须同时提及：
- ✅ "收盘小红，但盘后画风突变——XX暴跌X%，明天开盘要警惕"
- ❌ "今天小幅上涨，不用慌"（忽略了盘后异动）

**4. 今日建议**：根据盘后异动调整：
- 盘后暴跌 → "明天开盘可能大跌，心理上做好准备"
- 盘后暴涨 → "明天可能高开，但不要追涨"
- 无异动 → 正常建议

**⚠️ 核心原则：日报反映"此刻最新状态"，不是"收盘那一秒的状态"。如果盘后发生了大事，收盘价就过时了。**

1. **不写政治新闻**

6. **不编造 KOL 发言**（搜不到标"近期观点"）
7. **大涨方向可以出现在雷达里，但必须评估是否值得追**（priced in 多少？风险回报比？）
8. **没有合格方向就写"今天没有"**，不硬凑
9. **持仓不显示 IBKR 账号和股数**
10. **资金流向卡片必须说明"聪明钱在买什么"**
11. **标普500和SPY不要同时出现**
12. **休市日复用上一交易日数据，标注"数据截至X月X日收盘"**
13. **每次生成日报后必须执行全部步骤**（生成→转换MD→更新首页→归档→发布）
14. **个股如出现在雷达里，必须包含完整财务数据**，缺数据就换方向
15. **⚠️ 必须搜索并报告盘后/盘前异动**（Step 1.6）。日报反映"此刻最新状态"而非收盘价。如果盘后期货偏离收盘>±1%或持仓个股盘后涨跌>±3%，必须在日报中突出显示并调整建议。只报收盘价而不报盘后异动 = 严重失职
16. **"今日一句话"必须综合收盘价和盘后走势两个信号**。如果盘后方向与收盘相反，必须同时提及两者
17. **绝不复用之前对话中的任何数据**。每次生成日报时，所有数据必须从零开始重新搜索
18. **机会雷达优先追踪已有方向（watchlist.md），不每天发明新机会**。每周最多引入1-2个新方向。大多数日子应该是"继续追踪+今日更新"。不是每天都需要新东西买。

## 🔭 机会雷达规则（内联在prompt中，不依赖外部文件）

### 核心理念
**好的机会不会每天出现。专业分析师维持一个"观察清单"持续追踪，而不是每天发明新机会。**

### 运行机制

生成日报前，尝试读取 `0. Newsletter\watchlist.md`（观察清单）。

- 如果存在 → 更新已有条目的最新状态，只发现真正新方向时才添加新条目
- 如果不存在或为空 → 尝试发现1个新方向，创建watchlist.md
- 同时更新 watchlist.md 文件（写入今天的更新内容）

### watchlist.md 格式

```markdown
# 🔭 观察清单

## [类型emoji] [方向名] — [状态]
- **类型**：🔄 周期定位 / 💎 被低估好公司 / 🌊 蓝海新技术
- **开始追踪日期**：YYYY-MM-DD
- **状态**：🟢新发现 / 🟡追踪中 / 🔵催化剂兑现 / ✅已兑现 / ❌失效
- **逻辑**：[为什么这个方向值得关注，具体数据]
- **催化剂**：[等待什么事件来确认]
- **更新**：YYYY-MM-DD [今天有什么变化]
- **怎么参与**：[ETF选项 / 个股（需完整财务数据）]
```

### 状态标签

| 状态 | 含义 |
|---|---|
| 🟢 新发现 | 刚加入清单，需要更多数据验证 |
| 🟡 追踪中 | 逻辑成立，等催化剂兑现 |
| 🔵 催化剂兑现 | 等待的信号已出现，升级为"值得行动" |
| ✅ 已兑现 | 机会已反映在价格中，可以移除 |
| ❌ 逻辑失效 | 基本面恶化或判断错误，标记原因后移除 |

### 引入新方向的门槛（严格）

每周最多引入1-2个新方向。新方向必须满足以下全部条件：

**必须回答三个问题：**
1. **为什么是现在？**（不是上周，不是下个月，为什么今天加入观察清单）
2. **数据信号是什么？**（具体哪个数字变了，不是"感觉值得关注"）
3. **和读者已有的持仓有什么不同？**（如果VOO/QQQ里已经有了，不配进清单）

**必须满足至少一个引入依据：**

| 依据 | 合格示例 | 不合格示例 |
|---|---|---|
| A. 数据拐点 | "德州仪器Q1营收+19%，连续2季度加速，库存天数从80天降到55天" | "半导体行业可能复苏" |
| B. 事件创造新机会 | "特朗普限制GPT-5.6→AI监管可能利好隐私计算赛道" | "AI概念很热" |
| C. 估值错配 | "迪士尼跌30%但Disney+用户增速+8%，P/E 15x低于5年均值22x" | "迪士尼跌了不少" |

**必须排除的假信号：**
- ❌ 今天暴涨的股票（市场已经知道了，进场是接盘）
- ❌ 某分析师说"强烈推荐"（没有自己的数据判断）
- ❌ "长期来看XX一定涨"（没有时间窗口，等于什么都没说）
- ❌ 已经在VOO/QQQ里占大权重的方向（你已经有了）
- ❌ 上周刚推过然后移除的（死了的方向不要复活，除非有新数据）

### 每日日报里机会雷达板块的写法

**情况1：有正在追踪的条目（最常见）**
```
🔭 机会雷达

🔄 [方向名]
状态：🟡 追踪中（第X天）
今日更新：[今天的新闻/数据对这个方向有什么影响]
（如果今天没有新信息影响它，写"今日无新变化，继续追踪"）
```

**情况2：今天发现了新机会**
```
🔭 机会雷达

🟢 新方向：[方向名]
[完整分析：为什么值得关注/数据信号/怎么参与]
→ 已加入观察清单

🔄 [已有条目]
状态：🟡 追踪中
今日更新：...
```

**情况3：没有新发现，追踪的条目也无变化**
```
🔭 机会雷达

今日无新方向。正在追踪：
🔄 [方向1] — 🟡 追踪中（第X天）
📊 [方向2] — 🟡 追踪中（第X天）

耐心等待催化剂。不是每天都需要新机会。
```

### 什么时候移除条目

| 情况 | 操作 |
|---|---|
| 催化剂已兑现 | 标记🔵→✅，保留1周后移除 |
| 逻辑失效 | 标记❌，写明原因，保留1周后移除 |
| 追踪超过3个月仍无催化剂兑现 | 评估是否值得继续，可能需要移除 |

## 触发时间建议

| 时令 | 建议触发时间（北京时间） | 原因 |
|---|---|---|
| 夏令时（3月中-11月初） | 8:00 | 美东20:00=北京8:00，盘后交易刚好结束 |
